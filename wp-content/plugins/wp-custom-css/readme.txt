== Description ==
= Add Custom CSS Code To Your WordPress Site =

This plugin allows you to customize your WordPress site's appearance by enabling you to easily add custom CSS code to your site without ever having to modify your theme or plugin files.
The WP Custom CSS plugin has an easy to use interface which accepts CSS code via the WP admin panel settings page and applies the code to your site automatically.

== Installation ==

1. Upload the 'wp-custom-css.zip' file from the Plugins->Add New page in the WordPress administration panel.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings menu under 'WP Security' and start activating the security features of the plugin.

== Usage ==

Go to the settings menu after you activate the plugin and follow the instructions.
